# four.meme-indexer

## Instructions:

Use node 16\
\
Install packages:\
`npm install`

`npm run build`\
\
`npm run start`\
\
GET: `curl http://localhost:3001/latest` \
or\
POST: `curl -X POST -H "Content-Type: application/json" -d'{"name":"PET","address":"0x83a5298f921cc42513f497cd9c4b6194e6e94444"}' http://localhost:3001/latest`
